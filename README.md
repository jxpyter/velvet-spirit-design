# Velvet Spirits

**Velvet Spirits** is an ultra-premium, immersive e-commerce experience designed for the connoisseur of fine spirits. Built with a focus on editorial design, cinematic animations, and a "luxury first" aesthetic.

## üåü Features

*   **Immersive Editorial Design**: Full-screen, high-fidelity imagery with rich typography (`Allura`, `Cinzel`, `Playfair Display`).
*   **Cinematic Animations**:
    *   Staggered text reveals with blur effects.
    *   Ambient background lighting and localized spotlight effects.
    *   Smooth page transitions and scroll interactions powered by **GSAP**.
*   **Minimalist Commerce**:
    *   Seamless "Add to Bag" experience without page reloads.
    *   State-based, animated Checkout flow (Processing -> Success).
    *   Glassmorphism UI elements (frosted glass cards).
*   **Performance Optimized**: Hardware-accelerated animations (`will-change`), eager image loading, and efficient component rendering.

## üöÄ Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/velvet-spirits.git
    cd velvet-spirits
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
